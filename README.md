# Portfolio

Personal portfolio of **Lise Rochat**, freelance web developer focused on **minimal, fast and eco-designed websites**.

This project is a simple static website built with **HTML and CSS only**, designed to be lightweight, accessible and easy to maintain.

## Purpose

- Present my profile and services  
- Showcase my eco-design approach  
- Provide a lightweight professional portfolio  

## Technologies

- HTML5  
- CSS3  

## Deployment

Deployed with **GitHub Pages**:  
https://lise-dev.github.io/portfolio/

## Eco-design principles

- No external libraries
- No frameworks
- System font stack
- Inline SVG icons
- Optimized, lightweight code
